## Technologies
Project is created with:
* Vite
* React
* React Router DOM
* React Helmet Async
* React Bootstrap
* JWT Authentication
* PayPal.js
* MongoDB
* Express
* Node.js

To run this project, install it locally using npm:

```

$ npm run build
```
